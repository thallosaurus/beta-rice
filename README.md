# beta-rice
use `mount-config.sh` to temporarily mount the config.
use `unmount-config.sh` to unmount it.

## dependencies
- pywal16
- ranger (for wallpaper picker)
- waybar
- swayfx
- wofi
- caja

## setup
clone the repo and fetch the submodules for the wallpapers. copy the contents of the config folder to your `.config` and restart sway
